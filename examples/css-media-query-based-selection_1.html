CSS media query-based selection - 1

Example 1

The image has a flexible height and width. In total, 3 different layouts are applied:

  1. Breakpoint 1: Narrow
    Viewport width < 25rem), image width is 100px*.
  2. Breakpoint 2: Medium
    Viewport width < 50rem), image width is 200px*.
  3. Breakpoint 3: Wide
    Viewport width > 50rem image width is 300px*.
*px refers to CSS pixels.

User agents (e.g. browsers) cannot choose the appropriate resolution image, depending on the screen's pixel density, only one source per media query is provided.
