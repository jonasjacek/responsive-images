Responsive Image - 1

This responsive image uses a srcset with 3 different sources and reacts to viewport width.

sizes

The user agent will pick a width from the sizes attribute, using the first item with a <media-condition> (the part in parentheses) that evaluates to true, or using the last item (max-width: 450px) if they all evaluate to false.

Fallback

Browsers which do not support the srcset attribute will ignore it. Instead, they will load the image referenced in the src attribute.

Support can be looked up at https://caniuse.com/srcset.
