Viewport-based selection

Example 2

The image has a flexible height and width. In total, 3 different layouts are applied:

  1. Layout 1: Narrow
    Viewport width < 25rem), image takes the full width (100vw) of the available screen
  2. Layout 2: Medium
    Viewport width < 50rem), image takes half the width (50vw) of the available screen
  3. Layout 3: Wide
    Viewport width > 50rem, image takes roughly ⅓ (33vw) of the the available screen.

The user agent (e.g. browser) picks a width from the sizes attribute, using the first item with a so called <media-condition> (in parentheses) that evaluates to true, or uses the last item 33vw if all previous <media-condition> evaluate to false.

The available screen includes document margins. Document margings have been set to 0 in this example to prevent horizontal scrolling.

User agents (e.g. browsers) calculate the effective pixel density of each image by using the srcset's w-descriptors.
